# Analisis_de_comportamiendo_del_cliente
Proyecto de analisis de Datos del comportamiento del cliente usando Python/SQL/Power Bi
# 📊 Data Analytics Project

## Descripción general (Overview)

Este proyecto presenta un flujo completo de análisis de datos de principio a fin (End-to-End). El proceso incluye la carga y exploración de datos utilizando Python, la limpieza y transformación de la información, el análisis mediante consultas SQL, la creación de un dashboard interactivo en Power BI y la presentación de los hallazgos mediante un informe y una presentación en Gamma.

---

## Conjunto de datos (Dataset)

El conjunto de datos contiene información relevante para realizar un análisis exploratorio y extraer conocimientos útiles para la toma de decisiones.

Las actividades realizadas incluyen:

* Carga y revisión inicial de los datos.
* Identificación de valores faltantes y registros duplicados.
* Corrección de formatos y tipos de datos.
* Preparación de la información para su posterior análisis.

---

## Herramientas (Tools)

Las tecnologías utilizadas en este proyecto son:

* **Python:** Carga, limpieza, transformación y análisis de datos.
* **Pandas y NumPy:** Manipulación y procesamiento de datos.
* **Matplotlib y Seaborn:** Creación de visualizaciones exploratorias.
* **SQL (PostgreSQL, MySQL o SQL Server):** Consultas y extracción de información clave.
* **Power BI:** Desarrollo de dashboards interactivos y visualizaciones de negocio.
* **Gamma:** Diseño de la presentación final y storytelling de los resultados.

---

## Pasos del proceso (Steps)

### 1. Carga de datos

Se importó el conjunto de datos en Python y se realizó una revisión de su estructura, dimensiones y calidad.

### 2. Análisis exploratorio de datos (EDA)

Se analizaron estadísticas descriptivas y se generaron visualizaciones para identificar tendencias, patrones, anomalías y posibles problemas en los datos.

### 3. Limpieza y transformación de datos

Se preparó el conjunto de datos mediante:

* Eliminación de registros duplicados.
* Tratamiento de valores faltantes.
* Corrección de inconsistencias.
* Transformación y creación de nuevas variables.

### 4. Análisis con SQL

Se realizaron consultas en PostgreSQL, MySQL o SQL Server para responder preguntas de negocio y obtener información relevante.

### 5. Desarrollo del Dashboard en Power BI

Se creó un dashboard interactivo con:

* Indicadores clave de rendimiento (KPIs).
* Tendencias y comparaciones.
* Gráficos dinámicos.
* Principales hallazgos del análisis.

### 6. Informe y presentación en Gamma

Los resultados obtenidos se documentaron en un informe profesional y se resumieron en una presentación visual enfocada en comunicar insights de negocio.

---

## Dashboard

El dashboard en Power BI permite explorar los datos de manera interactiva y comprender rápidamente los indicadores más importantes, facilitando una toma de decisiones basada en datos.

## Resultados (Results)

El análisis permitió identificar patrones, tendencias y oportunidades de mejora dentro del conjunto de datos.

Principales resultados:

* Descubrimiento de insights relevantes.
